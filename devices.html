add_device(device_id, device_kind)

Add the specified device to the network.

add_input(device_id, input_id)

Add the specified input to the specified device.

Return True if successful.

add_output(device_id, output_id, signal=0)

Add the specified output to the specified device.

Return True if successful. The default output signal is LOW (0).

cold_startup()

Simulate cold start-up of D-types and clocks.

Set the memory of the D-types to a random state and make the clocks begin from a random point in their cycles.

find_devices(device_kind=None)

Return a list of device IDs of the specified device_kind.

Return a list of all device IDs in the network if no device_kind is specified.

get_device(device_id)

Return the Device object corresponding to device_id.

get_signal_ids(signal_name)

Return the device and output IDs of the specified signal.

get_signal_name(device_id, port_id)

Return the name string of the specified signal.

The signal is specified by its device_id and port_id. Return None if either ID is invalid.

make_clock(device_id, clock_half_period)

Make a clock device with the specified half period.

clock_half_period is an integer > 0. It is the number of simulation cycles before the clock switches state.

make_d_type(device_id)

Make a D-type device.

make_device(device_id, device_kind, device_property=None)

Create the specified device.

Return self.NO_ERROR if successful. Return corresponding error if not.

make_gate(device_id, device_kind, no_of_inputs)

Make logic gates with the specified number of inputs.

make_switch(device_id, initial_state)

Make a switch device and set its initial state.

set_switch(device_id, signal)

Set the switch state of the specified device to signal.

Return True if successful.
